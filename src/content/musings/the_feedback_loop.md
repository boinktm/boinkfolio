Iteration is not cleanup. Iteration is discovery.

Most of my stalled projects came from trying to force a final-quality pass too early. Now I split work into short loops:

1. Ship a rough pass quickly.
2. Identify one failure mode.
3. Solve only that failure.
4. Repeat.

Creative blocks usually show up when the next step is too large. The antidote is shrinking scope until the next action is obvious.

An unfinished piece is not a failure state. It is just a snapshot in the loop.
